ImageTrans

Computer-aided image and comic translation tool. Automatically detect, OCR, erase, and re-render text in images — designed for comics, manga, posters, and more.

Features

🔍

Accurate Text Detection

Natural scene text detection, object detection, and a heuristic method specially designed for comic speech bubbles. Handles complex layouts.

📖

Multi-Engine OCR

Support for multiple OCR engines with custom plugin support. Major world languages. Vertical text recognition for Japanese and Chinese. Layout analysis for reading order. Use LLMs to proofread and improve OCR accuracy.

🌐

Translation

Machine Translation: multiple MT engines and AI large language models (ChatGPT, Gemini, DeepSeek, etc.). Context-aware translation using surrounding text or source image via vision models.

Assist Tools: side-by-side comparison of multiple results, translation memory, corpus concordance, term management, furigana and pinyin annotation, and a sound effects search engine for comics.

🧹

Text Removal

Binarization, PatchMatch, Lama Inpaint, and Gemini Nano Banana for clean text removal. Preserve screentones when erasing text.

🎨

Text Typesetting

  • Auto-adjusted font sizing
  • Optimized line-breaking for space-separated languages (e.g. Japanese→English)
  • Smart punctuation positioning
  • Rich text with font styles (like Word's style system)
  • Text color detection
  • Rotation detection
  • Vertical text layout
🖌️

Export to PSD

Save results as Photoshop PSD files with separate text layers, ready for further editing by designers.

📦

Batch Processing

Process hundreds of images in one go with customizable operation pipelines, command-line invocation, and multi-process parallel processing for high-speed throughput.

🖥️

Intuitive & Powerful UI

Visual editor with real-time preview. Step through OCR, translation, image editing, lettering, search & replace, custom workflows, and more — all within a unified interface. Powerful yet easy to use.

Extra Features

📄

PDF Translation & Markdown

Translate PDFs and convert to Markdown — output original, translated, or bilingual — with layout analysis for multi-column papers, formulas, tables, and footnotes. Generate searchable PDFs from scanned documents.

🖥️

Real-Time Screen Translation

Monitor a screen region, auto-detect content changes, OCR and translate in real time with translation memory support.

🎬

Hardcoded Subtitle Extraction

Extract hardcoded subtitles from videos with fast OCR detection — high-speed and accurate. Output to SRT subtitle files.

📷

Document Scanning & Enhancement

Integrated TWAIN / WIA / SANE / ICA / eSCL scanning support. Clean, deskew, and enhance scanned books for optimal image preprocessing.

Usability

⌨️

Keyboard Shortcuts

Nearly every operation is accessible via keyboard shortcuts, enabling fast and efficient workflows without reaching for the mouse.

💻

Command-Line & API

Invoke ImageTrans from the command line for automated tasks. Run as an API server to serve translation requests over HTTP, enabling integration with any application or service.

⚙️

Customizable Workflows

Configure and chain processing steps — detection, OCR, translation, rendering — to build automated workflows that match your specific needs.

🤖

AI Agent Integration

Let AI agents call ImageTrans via ImageTrans Skills — structured skill definitions that enable LLM-powered agents to drive translation tasks programmatically.

📐

Smart Alignment & Search

Alignment guides when moving text boxes, batch-align selected boxes to source positions, and built-in search & replace. Well suited for e-commerce image translation.

Cross-Platform & Low Requirements

💻

Windows · macOS · Linux

Runs on Windows 7+, macOS 10.11+, and Linux. Can also serve as a server for online access from mobile devices.

⚡

Lightweight

~400 MB including Java runtime and models. Works on older or low-end devices. Works offline on your local machine.

🔌

Interoperable

PSD export, Excel/Word/XLIFF data exchange, LabelPlus project conversion, and Chrome Extension for web page translation.
