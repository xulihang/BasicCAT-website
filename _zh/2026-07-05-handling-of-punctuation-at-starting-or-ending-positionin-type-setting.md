文字排版中的标点需要避头尾。就是有些标点符号不能放在一行的开头或者结果。

比如左引号不能放在句尾，句号不能放在开头。

怎么处理两端的标点有很多的方式。有像上一行借字、标点悬挂、压缩间距等等。小时候在方格纸上写作文，一般标点如果要写到下一格开头了，我都是写在最后一个文字的格子里。

## 例子

原文：

```
在学术工作中，我们可以会遇到一些外语论文
。英语可能还好，但读起来不如中文母语效率
高。
```

标点悬挂：

```
在学术工作中，我们可以会遇到一些外语论文。
英语可能还好，但读起来不如中文母语效率高。
```

向上一行借字：

```
在学术工作中，我们可以会遇到一些外语论
文。英语可能还好，但读起来不如中文母语
效率高。
```

Word等字处理软件、Photoshop等图像编辑软件还有浏览器都支持这种处理。JavaFX这样的图形库也支持。

漫画翻译软件[ImageTrans](/zh/imagetrans/)使用了JavaFX排版文字，但竖排引擎是我自己实现的。目前初步支持了避免句号出现在行首。
