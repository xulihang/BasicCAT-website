如果你看不懂日语，那么在 iPhone 上阅读日漫（Manga）和同人志（Doujinshi）可能会非常困难。虽然现代浏览器能够翻译网页文本，但通常无法翻译嵌入在图片中的文字。

而这恰恰是一个常见问题，因为漫画、同人志、连环画、扫描书籍、截图以及许多社交媒体内容，绝大部分信息都是以图片形式呈现的。

借助 ImageTrans 浏览器插件和 Orion Browser，你可以直接在 iPhone 上翻译图片中的文字。

## 为什么传统网页翻译器不够用？

Google Translate 等内置翻译工具主要针对 HTML 文本进行翻译。

然而，许多网站的内容实际上是图片：

* 漫画页面
* 同人志
* 连环画
* 扫描书籍
* 截图
* 信息图（Infographics）
* 社交媒体图片

由于文字本身已经嵌入到了图片中，普通网页翻译器无法读取和翻译这些内容。

ImageTrans 通过对图片进行 OCR 识别、翻译以及图像编辑，解决了这一问题。

## 为什么选择 Orion Browser？

苹果的 Safari 浏览器并不直接支持 Chrome 扩展。

幸运的是，Orion Browser 在 iPhone 和 iPad 上提供了对大量 Chrome 和 Firefox 扩展的实验性支持。

因此，我们可以在 iOS 上安装并使用 ImageTrans 浏览器插件。

安装完成后，ImageTrans 就可以帮助你在浏览网页时翻译漫画页面、扫描文档以及其他基于图片的内容。

## 两种图片翻译方式

### 方式一：图片翻译

插件会检测网页中的图片，并将其发送给 ImageTrans 进行处理。

处理流程如下：

1. 检测图片。
2. 使用 OCR 提取文字。
3. 翻译文字。
4. 清除原始文字。
5. 渲染译文。
6. 用翻译后的图片替换原图。

这种方式适用于：

* 漫画
* 同人志
* 连环画
* 扫描文档
* 海报
* 信息图

### 方式二：截图翻译

目前在 iPhone 上，截图翻译是最可靠的方案。

与直接下载网页图片不同，你只需要选择屏幕上的一个区域，然后将截图发送给 ImageTrans。

处理流程如下：

1. 截取选定区域。
2. 使用 OCR 识别文字。
3. 翻译文字。
4. 生成翻译后的图片。
5. 显示结果。

即使网页图片无法被直接访问，这种方式依然有效。

## iOS 上的当前限制

由于 Orion Browser 的扩展运行机制与桌面版 Chrome 不同，因此存在一些限制。

### 跨域图片访问

许多网站会将图片托管在与网页不同的域名下。

在桌面版 Chrome 中，ImageTrans 可以利用 DeclarativeNetRequest 规则修改请求，从而绕过部分跨域限制。

但在 iOS 版 Orion Browser 中，这些请求修改并不总是能够正常工作。

因此：

* 某些图片无法被插件下载。
* 直接图片翻译可能失败。
* 截图翻译通常是更可靠的替代方案。

对于漫画和同人志阅读场景来说，截图翻译通常是最稳定的选择。

### 使用本地 ImageTrans 服务

ImageTrans 支持在本地电脑上运行 OCR 和翻译服务，无需依赖云端平台即可处理图片。

在桌面浏览器中，插件通常通过 background script 与本地 ImageTrans 服务通信。这样可以避免访问局域网服务时遇到证书和 Mixed Content（混合内容）限制。

然而，在 iOS 版 Orion Browser 中，这种通信方式并不可靠。

因此，插件往往需要直接与 ImageTrans 服务通信。

由于如今大多数网站都运行在 HTTPS 下，浏览器会阻止 HTTPS 页面访问局域网中的 HTTP 服务。

为了正常通信，ImageTrans 服务也需要启用 HTTPS。

如果是在局域网环境中使用，则需要使用自签名证书。配置步骤如下：

1. 为 ImageTrans 服务生成包含服务器 IP 地址的证书。
2. 将证书安装到 iPhone。
3. 在 iOS 设置中将该证书标记为受信任。

完成后，浏览器便可以通过局域网 IP 安全地访问 ImageTrans 服务。

### 无法使用本地 PaddleOCR

ImageTrans 浏览器插件原本支持通过 WebAssembly（WASM）直接在浏览器内运行 PaddleOCR。

这样 OCR 可以完全在用户设备上执行，而无需额外服务器。

然而，目前 Orion Browser 在 iOS 上对相关 WASM 模块的加载存在兼容性问题。

因此，内置的 PaddleOCR 无法正常初始化。

受此限制影响，OCR 处理必须由运行在其他设备上的 ImageTrans 服务完成，例如：

* Windows
* Mac
* Linux

插件会将截图或图片发送到服务器，由服务器完成 OCR、翻译以及图片渲染，然后返回最终结果。

## 推荐配置方案

为了获得最佳的 iPhone 漫画和同人志阅读体验，推荐如下配置：

1. 安装 Orion Browser。
2. 安装 ImageTrans 浏览器插件。
3. 在 PC 或 Mac 等家庭电脑上上运行 ImageTrans。
4. 为 ImageTrans 服务配置 HTTPS 证书。
5. 在 iOS 上安装并信任该证书。
6. 在插件中配置 ImageTrans 服务地址。
7. 当图片翻译不可用时，优先使用截图翻译。

该方案能够规避当前 iOS 在跨域图片访问、扩展通信以及基于 WebAssembly 的 OCR 方面的限制。

**PS：**

如果你暂时不想部署 ImageTrans 服务，也可以先试用公共服务器。

不过由于公共服务器速度较慢，因此不建议用于大量或长期使用。

## 可以翻译哪些内容？

ImageTrans 并不仅限于漫画和同人志。

你还可以翻译：

* 连环画
* Webtoon
* 轻小说插图
* 游戏截图
* 日文博客
* 中文漫画
* 韩漫
* 扫描书籍
* 技术图纸
* 信息图
* 社交媒体图片

任何包含文字的图片，都有机会被自动翻译。

## 总结

虽然 iPhone 上的浏览器扩展支持仍然不如桌面平台完善，但 Orion Browser 已经让图片翻译成为可能。

通过 Orion Browser 与 ImageTrans 的组合，你可以在浏览网页时直接翻译漫画、同人志、连环画、扫描文档以及其他图片内容。

受当前 iOS 平台限制影响，截图翻译仍然是最稳定可靠的工作方式。而配合启用了 HTTPS 的 ImageTrans 服务后，你就能够在 iPhone 上更加流畅地阅读各种外语图片内容，而无需频繁切换应用。
