ImageTrans 浏览器扩展

一键翻译网页上的任何图片，专为漫画、海报、截图等场景设计。右键点击或使用快捷键即可获得翻译结果。

🌐 添加到 Chrome 🦊 添加到 Firefox

什么是ImageTrans浏览器扩展？

ImageTrans浏览器扩展将AI驱动的图片翻译能力直接带入您的浏览器。它可以检测网页中图片（漫画、海报、截图、宣传册等）上的文字，并自动将其替换为翻译后的版本。

基于 PaddleOCR 和 ImageTrans 桌面应用后端，该扩展将最先进的OCR技术与神经机器翻译相结合，为您提供准确、符合上下文的翻译结果。

主要功能

🖱️

右键翻译

在网页上右键点击任意图片，选择"翻译图片"即可立即获取翻译后的版本。

⌨️

键盘快捷键

按下 Ctrl+Shift+F 即可翻译光标下的图片，快速高效。

🔄

自动翻译

启用自动翻译后，页面加载时图片会自动翻译。非常适合阅读长篇漫画。

📸

截图翻译

在浏览器中截取屏幕区域进行翻译，适合翻译无法下载的图片或仅翻译图片的局部区域。

🤖

灵活的翻译选项

将任意OCR后端（ImageTrans或PaddleOCR）与OpenAI兼容API（ChatGPT、Gemini、DeepSeek）或内置翻译引擎搭配使用，获得最佳翻译效果。

🀄

注音标注

为日语文字标注假名，为中文文字标注拼音——语言学习者的得力助手。

如何使用

  1. 安装扩展 从 Chrome应用商店 或 Firefox附加组件 添加。
  2. 配置后端 连接到 ImageTrans 服务器，使用内置PaddleOCR，或配置您自己的OpenAI兼容API密钥。
  3. 开始翻译 右键点击图片 → "翻译图片"，或对任意图片按 Ctrl+Shift+F。完成！

工作原理

🔌 配合 ImageTrans 桌面版（推荐）

  • ImageTrans 负责完整流程：检测 → OCR → 文字抹除 → 翻译 → 渲染
  • 最佳的准确度、最广的语言支持和丰富的编辑功能
  • 扩展将图片发送给 ImageTrans 处理后直接展示成品
  • 需要 ImageTrans 在本地或服务器上运行

🧩 不依赖 ImageTrans（纯浏览器运行）

  • 扩展在前端独立完成全部流程：OCR（PaddleOCR）→ 翻译 → 渲染
  • 无需服务器或安装任何软件——完全自包含
  • 免费且保护隐私
  • 准确度、速度和语言选项较为有限

翻译选项

以下翻译方式在两种模式下均可使用。当配合 ImageTrans 使用 OpenAI 兼容 API 时，ImageTrans 仅负责 OCR，翻译和渲染由扩展在前端完成。

🌐 ImageTrans 内置翻译

  • 连接 ImageTrans 时可用
  • 支持多种翻译引擎（Google、DeepL、百度等）
  • 包含翻译记忆、术语管理和语料检索功能

🤖 OpenAI 兼容 API

  • 使用 ChatGPT、Gemini、DeepSeek 或任何兼容 API
  • 配合或不配合 ImageTrans 均可使用——使用自己的 API 密钥
  • 灵活的自定义提示词，实现更细致的翻译
  • 配合 ImageTrans 时：ImageTrans 仅负责 OCR，扩展负责翻译和渲染

🆓 MyMemory（免费）

  • 免费机器翻译 API，无需密钥
  • 适合快速、临时的翻译需求

关于 ImageTrans 桌面版

ImageTrans 是一款功能完备的计算机辅助图片翻译桌面软件。浏览器扩展适合快速即时翻译，而 ImageTrans 桌面版则让您完全掌控整个翻译工作流程。

🎛️

强大的定制功能

精细调整每个步骤：从多种OCR引擎中自由选择，配置检测算法，自定义渲染参数，精确满足您的需求。

✏️

手动编辑

调整文字区域、修正OCR结果、编辑翻译内容、微调文字位置——当自动化不够时，一切尽在掌握。

🎨

导出 PSD

将翻译后的图片导出为 Photoshop (PSD) 文件，文字分层独立，方便交给设计师进行后期处理。

📦

批量处理

一次性翻译数百张图片。支持通过 Excel、Word 或 XLIFF 导入导出数据，与其他翻译工具无缝对接。
