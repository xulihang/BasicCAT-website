ImageTrans

计算机辅助图片翻译工具。自动检测、识别、抹除并回填图片中的文字——针对漫画、PDF文档等场景优化。

核心功能

🔍

准确定位文字区域

支持自然场景文字检测、目标检测和启发式等多种方法，拥有针对漫画对话框设计的文字区域合并与检测算法。

📖

多引擎 OCR 识别

支持多种 OCR 引擎及自定义插件，覆盖世界主要语言。支持竖排中日文识别，通过版面分析确定阅读顺序。可利用大语言模型校对改善 OCR 准确度。

🌐

翻译辅助

支持多种机器翻译引擎和 AI 大模型（ChatGPT、Gemini、DeepSeek 等）并排对比结果。利用上下文信息提升翻译质量。配备翻译记忆、语料检索、术语管理，以及漫画拟声词检索系统。

🎨

文字抹除与回填

支持二值化、PatchMatch、Lama Inpaint、Gemini Nano Banana 等算法和 AI 模型生成文字掩膜并去除文字。去除文字时保留网点纸。自动调整译文字体大小，标点避头尾处理，支持竖排文字排版。全部支持手动调整。

🖌️

导出 PSD

将结果保存为 Photoshop PSD 文件，文字分层独立，方便设计师进行后期处理。

📦

批量处理

一次性处理数百张图片。全局字体样式、背景色/文字色检测、旋转方向检测、智能排序等完备功能。

更多功能

📄

PDF 翻译与 Markdown 转换

翻译 PDF 并转换为 Markdown，可输出原文、译文或双语对照，保留多栏论文、公式、表格和脚注等结构。生成可搜索的双层 PDF。

🖥️

屏幕实时翻译

实时捕获并翻译屏幕任意区域。自动检测内容变化，支持翻译记忆即时匹配。

🎬

视频硬字幕提取

提取视频中的硬字幕，利用快速 OCR 文字检测，高速又准确，输出 SRT/ASS 字幕文件。

📷

文档扫描与增强

支持 TWAIN / WIA / SANE / ICA / eSCL 集成扫描。清理、纠偏和增强扫描书籍，获得最佳图像预处理效果。

跨平台 · 低硬件要求

💻

Windows · macOS · Linux

支持 Windows 7+、macOS 10.11+ 和 Linux。亦可作为服务器运行，通过移动设备扫描和翻译图片。

⚡

轻量高效

包含 Java 运行环境和模型约 400 MB，可在较旧或低端设备上运行。支持离线工作，无需联网。

🔌

互操作性

PSD 导出、Excel/Word/XLIFF 数据交换、LabelPlus 项目转换，配合 Chrome 扩展翻译网页中的图片。
