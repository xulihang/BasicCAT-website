BasicCAT

BasicCAT is a free and open source computer-aided translation tool. We can use it to translate documents, web pages, tables, slides and so on with features like translation memory, machine translation and terminology management.

ImageTrans

ImageTrans a computer-aided image translation tool. We can use it to translate comics, manga, posters, etc.

Aligner
